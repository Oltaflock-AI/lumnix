## Aesthetic Direction: "The War Room"

**Concept:** Marketing intelligence isn't a dashboard — it's a command centre. The user is a general, not a guesser. Dark, editorial, military-intelligence-meets-luxury-tech. Think Bloomberg Terminal crossed with a high-end editorial magazine. The one thing people will remember: *it feels like gaining altitude on your competition.*

**NOT:**
- Purple gradients on white — generic AI slop
- Floating 3D screenshots of a dashboard — everyone does this
- "Powered by AI" headlines — meaningless in 2026
- Hero with smiling stock photo people
- Pastel "friendly SaaS" vibes

**YES:**
- Near-black canvas `#07051A` with deep indigo undertone
- Typography as the primary visual element — massive, editorial, kinetic
- 3D interactive card effects that respond to mouse movement
- Real numbers as design elements (not icons)
- Copy that punches you in the stomach with specificity
- A "live data" section that simulates the product working in real time

## Psychological Copy Framework

### Core Pain: Tool Fragmentation + Decision Paralysis

The target is an Indian D2C founder or agency marketer. Their daily reality:
- Open GSC at 9am → switch to GA4 → export to Sheets → paste into Ads Manager → repeat
- Paying ₹15,000–75,000/mo across tools that never talk to each other
- Making creative decisions based on vibes because the data is too scattered to read
- Watching competitors run ads for months and not knowing why they work

### Psychological triggers to deploy:

**1. Specificity as credibility** — Don't say "save time". Say "stop exporting CSVs at 11pm before a client call."

**2. Loss aversion** — "Every day you don't know which ads your competitors are running is a day they're stealing your customers."

**3. Social proof through data** — Real-feeling metrics that validate the tool (even during beta: "₹2.4 crore in ad spend analyzed")

**4. Identity shift** — Not "a better dashboard". "The difference between guessing and knowing."

**5. The villain reframe** — The enemy is fragmented tools, not incompetence. "It's not that you're bad at marketing. It's that your tools were built to be used separately."

### 3D Card Effect Implementation

```typescript
// Mouse-tracking 3D tilt effect
// Apply to any card with: <Card3D>...</Card3D>

const Card3D = ({ children }) => {
  const cardRef = useRef(null)
  const [transform, setTransform] = useState('')

  const handleMouseMove = (e) => {
    const card = cardRef.current
    const rect = card.getBoundingClientRect()
    const x = e.clientX - rect.left
    const y = e.clientY - rect.top
    const centerX = rect.width / 2
    const centerY = rect.height / 2
    const rotateX = ((y - centerY) / centerY) * -8  // max 8deg
    const rotateY = ((x - centerX) / centerX) * 8
    setTransform(`perspective(1000px) rotateX(${rotateX}deg) rotateY(${rotateY}deg) scale(1.02)`)
  }

  const handleMouseLeave = () => {
    setTransform('perspective(1000px) rotateX(0deg) rotateY(0deg) scale(1)')
  }

  return (
    <div
      ref={cardRef}
      onMouseMove={handleMouseMove}
      onMouseLeave={handleMouseLeave}
      style={{ transform, transition: 'transform 200ms ease' }}
    >
      {children}
    </div>
  )
}
```
